Network Protocols

Types and definitions:

File Transfer Protocol (FTP):
FTP is a set of rules that allow files to be exchanged between computers on the Internet. Web developers commonly use FTP to transfer web page files from their computers to web servers. FTP is also commonly used to download programs and files from other servers to individual computers.
E-mail Protocols:
When you send e-mail to others, Simple Mail Transfer Protocol (SMTP) is used. When you receive e-mail, Post Office Protocol (POP; currently POP3) and Internet Message Access Protocol (IMAP) can be used.
Hypertext Transfer Protocol (HTTP):
HTTP is a set of rules for exchanging files such as text, images, audio, video, and other multimedia on the Web. Web browsers and web servers usually use this protocol.
